### 👨‍💻 About Me

I am a **System and Network Engineer** based in Baku, Azerbaijan, currently working at **SEABAK**. My expertise lies in infrastructure automation, system standardization, and secure network management. I hold the **INE Certified Cloud Associate (ICCA)** and **RHCSA** certifications, and I am actively transitioning my career toward **DevOps and Cloud Engineering**.

- 🎓 **Certifications & Goals:** **RHCSA** & **ICCA** certified. Currently preparing for **RHCE** and **CKA** (Certified Kubernetes Administrator).
- 🏢 **Enterprise Experience:** Windows Server migrations (2012 R2 to 2022), Active Directory architecture, VMware, RHEL, AlmaLinux, and Dell PowerEdge hardware management.
- 🛡️ **Monitoring & Security:** Implementation of Wazuh for log management and extensive administration of Zabbix infrastructure.
- ⚡ **Automation:** Developing automated deployment kits using PowerShell and Batch to streamline daily operations.

---

### 🚀 Key Projects & Deployments

- **`ZabbixConfig`** - An automated tool developed to handle the silent installation and proxy IP configuration of Zabbix Agent 2 across corporate environments.
- **`SeabakWallpaperEngine`** - A custom executable designed to standardize and enforce corporate desktop backgrounds centrally via Group Policy (GPO).
- **`SysAdmin Deployment Kit`** - A comprehensive suite of PowerShell and Batch scripts engineered to accelerate infrastructure tasks and deployments.
